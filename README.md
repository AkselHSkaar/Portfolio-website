# My portfolio

My portfolio web page, made in [next.js](https://nextjs.org/), deployed to Azure Web App via Azure Container Registry.

Visit the [live site](https://akselskaar.no/)

## Project setup

```bash
pnpm install
```

```bash
pnpm run dev
```

Open [http://localhost:3000/](http://localhost:3000/).
